Tu es un développeur full-stack senior expert ERP, CRM, logistique, achats et comptabilité.

Objectif :
Créer un module FOURNISSEURS ultra professionnel inspiré des meilleurs ERP (Dolibarr, Odoo, SAP Business One),
adapté à une entreprise exportatrice de vanille à Madagascar.

Le module doit être :
- moderne
- responsive
- scalable
- relié à la comptabilité
- relié à la logistique
- relié au CRM/commercial
- relié aux achats
- prêt pour audit et gestion professionnelle

Stack obligatoire :
- Frontend : React + Tailwind + Shadcn UI
- Backend : Node.js + Express
- ORM : Prisma
- Database : PostgreSQL
- Upload : Multer
- OCR : Tesseract.js
- PDF : Puppeteer

IMPORTANT :
Le formulaire fournisseur doit gérer 2 TYPES DE FOURNISSEURS :

1. FOURNISSEURS DE BIENS
2. FOURNISSEURS DE SERVICES

Le formulaire doit adapter automatiquement les champs et catégories selon le type choisi.
