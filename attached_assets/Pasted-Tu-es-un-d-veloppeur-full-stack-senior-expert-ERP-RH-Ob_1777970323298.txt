Tu es un développeur full-stack senior expert ERP RH.

Objectif :
Créer un module DEMANDES RH professionnel avec :
- création de demandes (multi-types)
- workflow validation (manager + RH)
- génération de document imprimable (PDF)
- auto-remplissage des données
- export PDF et archivage

Stack :
- Backend : Node.js + Express
- DB : Prisma + PostgreSQL
- Front : React + Tailwind + React Hook Form
- PDF : Puppeteer
- Validation : Zod

-----------------------------------
1. MODÈLES PRISMA
-----------------------------------

model HRRequest {
  id            String   @id @default(uuid())
  employeeId    String
  type          String   // leave, advance, attestation, mission

  startDate     DateTime?
  endDate       DateTime?
  amount        Float?
  reason        String?

  status        String   // draft, pending, manager_approved, hr_approved, rejected

  managerId     String?
  hrId          String?

  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt
}

-----------------------------------

model HRRequestLog {
  id          String   @id @default(uuid())
  requestId   String
  action      String   // created, approved, rejected
  userId      String
  comment     String?
  createdAt   DateTime @default(now())
}
