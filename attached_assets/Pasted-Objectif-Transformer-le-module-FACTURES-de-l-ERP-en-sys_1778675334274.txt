Objectif :
Transformer le module FACTURES de l’ERP en système comptable ultra professionnel connecté à :
- achats logistique
- fournisseurs
- OCR intelligent
- comptabilité
- stock
- paiements

Stack obligatoire :
- Frontend : React + Tailwind + Shadcn UI
- Backend : Node.js + Express
- ORM : Prisma
- Database : PostgreSQL
- OCR : Tesseract.js + pdf-parse
- Upload : Multer
- PDF : Puppeteer

IMPORTANT :
Le système doit fonctionner comme un vrai ERP professionnel :
- génération automatique factures achats
- OCR intelligent
- liaison comptable automatique
- gestion pièces jointes
- permissions avancées
