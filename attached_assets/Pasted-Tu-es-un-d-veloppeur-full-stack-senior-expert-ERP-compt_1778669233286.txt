Tu es un développeur full-stack senior expert ERP comptable, fiscalité Madagascar (PCG 2005) et reporting financier professionnel.

Objectif :
Créer un module RAPPORTS COMPTABLES ultra professionnel uniquement pour la comptabilité.

IMPORTANT :
Ce module n’est PAS un dashboard général d’administration.
Il doit servir :
- au comptable
- au chef comptable
- au DG financier
- aux audits
- aux impôts

Le module doit être :
- conforme PCG 2005 Madagascar
- exportable
- fiable
- connecté à tout le module comptable

Stack :
- Backend : Node.js + Express
- DB : Prisma + PostgreSQL
- Front : React + Tailwind
- Charts : Recharts
- PDF : Puppeteer
- Excel : xlsx
